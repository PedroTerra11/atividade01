# 📱 Atividade 01 - Hello World em Dart

## 🎯 Objetivo

Este projeto tem como objetivo verificar a configuração do ambiente de desenvolvimento para a linguagem Dart, além de executar um programa simples no terminal.

Também tem como finalidade praticar o uso de Git e GitHub para versionamento de código.

---

## 🛠️ Tecnologias Utilizadas

* Dart
* Terminal/CLI
* Git e GitHub

---

## ▶️ Como Executar o Projeto

1. Certifique-se de ter o Dart instalado na máquina:

```bash
dart --version
```

2. Acesse a pasta do projeto:

```bash
cd seu-repositorio
```

3. Execute o programa:

```bash
dart run main.dart
```

---

## 💻 Exemplo de Saída

```text
Hello World
```

---

## 📌 Descrição do Código

O programa possui uma função principal chamada `main()`, que é o ponto de entrada da aplicação em Dart.

Dentro dela, utilizamos o comando `print()` para exibir a mensagem "Hello World" no terminal.

---

## 📎 Observações

Este projeto é uma atividade introdutória com foco na configuração do ambiente e execução de um programa simples em Dart.
